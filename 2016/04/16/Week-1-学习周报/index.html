学习笔记

  • WebGL Programming Guide 这本书，作者有一套独特的叙述WebGL的方法。先讲什么后讲什么是安排好的。比如作者会先搞一个库，屏蔽掉一些他认为不需要一上来就讲清楚的知识点，一步一步带着学习。这样的好处是事后你会发现自己是走了一条登山的捷径。坏处就是，一开始由于对webgl的不了解，你会不明白作者为什么要这么搞，一些东西被封装进了库，导致自己的好奇心没法得到满足。认真研读把这本书，收获还是非常大的。还需再读。
  • 利用Github Pages和Hexo搭建了一个博客，购买域名，优化DNS，学习到很多知识。
  • 通过准备OCJP考试，发现自己对于 Java SE 中有些不常用的知识点有所遗忘。需要多加巩固。
  • 尽快做完毕业设计，然后全身心投入到Web3D的学习中。
